# Proyecto Git - Tarea 1 MPO

## Descripción del Proyecto

Este proyecto es una tarea educativa para aprender Git y GitHub. Consiste en una página web básica en HTML que implementa un flujo de trabajo profesional con control de versiones, utilizando ramas y Pull Requests para desarrollar mejoras de manera organizada.
